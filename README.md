## 📁 Project Structure

```
Aqdi/
├── app/                    # Next.js App Router
│   ├── layout.tsx          # Root layout
│   ├── page.tsx            # Home page
│   └── globals.css         # Global styles
├── components/
│   ├── ContractForm.tsx    # Main form (Zod + react-hook-form)
│   ├── SignatureCanvas.tsx # Digital signature component
│   ├── ContractPreview.tsx # Live contract preview
│   └── ui/                 # shadcn-style components
├── lib/
│   ├── gemini.ts           # Gemini API integration
│   ├── pdf.ts              # pdf-lib export logic
│   └── schema.ts           # Zod validation schemas
├── __tests__/              # Vitest test suite
├── .env.example            # Required env variables
├── metadata.json           # AI Studio metadata
└── package.json
```

## 🚀 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/revolover00/Aqdi
cd Aqdi
git checkout Baldwin

# 2. Install dependencies
npm install

# 3. Set up environment
cp .env.example .env.local
# Add your Gemini API key 👇
```

### Environment Variables

```env
GEMINI_API_KEY=your_gemini_api_key_here
```

> Get your free API key from [Google AI Studio](https://aistudio.google.com/apikey)

```bash
# 4. Run locally
npm run dev
# → http://localhost:3000

# 5. Run tests
npm test
```
